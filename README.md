  <h1>User Authentication System</h1>

  <h2>Overview</h2>
  <p>
    This project is a user login and authentication system built using modern technologies such as <strong>MongoDB</strong>, 
    <strong>React</strong>, <strong>TypeScript</strong>, and <strong>Clerk</strong> for authentication. The system allows users to sign up, 
    log in, and manage personal information, which is securely stored in MongoDB.
  </p>

  <h2>Features</h2>
  <ul>
    <li><strong>User Authentication</strong>: Secure user login, sign-up, and logout functionality using Clerk.</li>
    <li><strong>User Profile Management</strong>: Each user has a unique profile where personal information is stored in MongoDB.</li>
    <li><strong>Tech Stack</strong>:
      <ul>
        <li><strong>React</strong> for front-end development and creating interactive user interfaces.</li>
        <li><strong>TypeScript</strong> for type safety and enhanced development experience.</li>
        <li><strong>MongoDB</strong> for storing user information and handling profile data.</li>
        <li><strong>Clerk</strong> for user authentication and session management.</li>
      </ul>
    </li>
  </ul>
